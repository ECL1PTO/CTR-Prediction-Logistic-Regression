# CTR Prediction using Logistic Regression

This repository contains a minor data science project that predicts whether a user will click on an online advertisement.

## Project Overview
- Model used: Logistic Regression
- Problem type: Binary classification (Click / No Click)
- Dataset: Online advertising data
- Evaluation metrics: Accuracy, Precision, Recall

## Files
- `CTR_Prediction_Logistic_Regression.ipynb` – main notebook
- `advertising.csv` – dataset used

## Objective
The goal of this project is to improve ad targeting by identifying users who are more likely to click on advertisements and reducing wasted ad spend.
